Política de Privacidade

Esta Política de Privacidade descreve como o sistema Conexão Design Gestão coleta, usa e protege as informações dos usuários.

1. Informações Coletadas

O sistema coleta apenas as informações necessárias para o funcionamento interno da empresa:

  • Nome e e-mail do usuário autenticado via Google OAuth
  • Dados financeiros e operacionais inseridos pelos próprios usuários
  • Registros de acesso para fins de auditoria interna

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para:

  • Autenticação e controle de acesso ao sistema
  • Gestão financeira e operacional da empresa Conexão Design Móveis e Decorações
  • Geração de relatórios internos

3. Compartilhamento de Dados

Os dados não são compartilhados com terceiros. O sistema é de uso exclusivo interno da empresa Conexão Design Móveis e Decorações LTDA - CNPJ 58.323.410/0001-53.

4. Armazenamento

Os dados são armazenados no Google Sheets, protegidos pelas políticas de segurança do Google. O acesso é restrito aos usuários autorizados pela administração da empresa.

5. Google OAuth

O sistema utiliza o Google OAuth apenas para autenticação. As permissões solicitadas são limitadas ao acesso à planilha de dados do sistema. Nenhuma informação pessoal da conta Google é armazenada além do e-mail para controle de acesso.

6. Seus Direitos

Os usuários podem solicitar a exclusão de seus dados de acesso a qualquer momento entrando em contato com o administrador do sistema.

7. Contato

8. Termos de Serviço

O uso deste sistema é restrito a colaboradores e administradores autorizados pela Conexão Design Móveis e Decorações. O acesso não autorizado é proibido. O sistema é fornecido para uso interno sem garantias de disponibilidade contínua.
